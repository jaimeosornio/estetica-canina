# 🐶 Sistema Estética Canina

Sistema web desarrollado con Angular y Spring Boot para la gestión de una estética canina.

## 🚀 Tecnologías utilizadas

- Angular
- TypeScript
- Bootstrap
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database

## 📋 Funcionalidades

### Mascotas
- Registrar mascotas
- Editar mascotas
- Eliminar mascotas
- Buscar por ID
- Buscar por nombre

### Clientes
- Registrar clientes
- Editar clientes
- Eliminar clientes
- Buscar por ID
- Buscar por nombre

### Responsables
- Registrar responsables
- Editar responsables
- Eliminar responsables
- Buscar por ID
- Buscar por nombre

## 📊 Dashboard

El sistema muestra:

- Total de mascotas registradas
- Total de clientes registrados
- Total de responsables registrados
